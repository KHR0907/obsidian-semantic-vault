# 피아노 연습법 과학적 접근

효과적인 피아노 연습은 양보다 질이 중요하다. 어려운 구간을 분리하여 느린 템포로 반복하는 블록 연습이 기본이다. 메트로놈을 사용하여 점진적으로 속도를 올리되, 실수 없이 세 번 연속 성공하면 속도를 높인다. 분산 연습(짧은 세션을 여러 번)이 집중 연습(긴 세션 한 번)보다 기억 정착에 효과적이다. 정신적 연습(악보를 보며 머릿속으로 연주)도 도움이 된다.
