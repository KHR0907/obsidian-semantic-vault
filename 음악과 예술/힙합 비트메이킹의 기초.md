# 힙합 비트메이킹의 기초

힙합 비트의 기본은 드럼 패턴이다. 킥은 1박과 3박에, 스네어는 2박과 4박에 배치하는 것이 기본이며, 하이햇은 8분이나 16분으로 쪼갠다. 808 베이스는 서브 저음으로 체감되는 진동을 만든다. 샘플링은 기존 음악의 일부를 잘라 재배치하는 기법이다. DAW에서 MIDI와 오디오를 조합하고, 사이드체인 컴프레션으로 킥과 베이스의 공간을 확보한다.
