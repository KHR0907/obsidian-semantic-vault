# 토큰화 전략 비교

토큰화는 텍스트를 모델이 처리할 수 있는 단위로 분할하는 과정이다. BPE(Byte Pair Encoding)는 가장 빈번한 문자 쌍을 반복적으로 병합하여 서브워드 사전을 구축한다. WordPiece는 BERT에서 사용되며 우도 기반으로 병합한다. SentencePiece는 언어에 독립적인 전처리를 제공한다. 한국어는 형태소 분석 후 토큰화하거나, 음절 단위 BPE를 적용하는 방법이 있다.
