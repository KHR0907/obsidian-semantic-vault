# LLM 프롬프트 엔지니어링

프롬프트 엔지니어링은 대규모 언어 모델에서 원하는 출력을 얻기 위해 입력을 설계하는 기술이다. 제로샷, 원샷, 퓨샷 프롬프팅이 기본이며, Chain-of-Thought는 단계별 추론을 유도한다. Tree of Thoughts는 여러 추론 경로를 탐색하고, ReAct는 추론과 행동을 결합한다. 시스템 프롬프트 설계는 모델의 전반적인 행동 방식을 결정하는 중요한 요소다.
