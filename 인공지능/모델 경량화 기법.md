# 모델 경량화 기법

모델 경량화는 대규모 모델을 모바일이나 에지 디바이스에서 실행할 수 있게 만드는 기술이다. 프루닝은 불필요한 가중치를 제거하고, 양자화는 가중치의 비트 수를 줄인다. 지식 증류는 큰 교사 모델의 지식을 작은 학생 모델에 전달한다. MobileNet과 EfficientNet은 경량 아키텍처의 대표적 사례다.
