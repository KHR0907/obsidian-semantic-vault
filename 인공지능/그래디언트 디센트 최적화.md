# 그래디언트 디센트 최적화

경사 하강법은 손실 함수의 기울기를 따라 모델 파라미터를 업데이트하는 최적화 기법이다. 배치 경사 하강법은 전체 데이터를 사용하고, 확률적 경사 하강법(SGD)은 샘플 단위로 업데이트한다. Adam 옵티마이저는 모멘텀과 적응적 학습률을 결합한 방법으로, 현재 가장 널리 사용되는 최적화 알고리즘이다. 학습률 스케줄링은 수렴 속도와 안정성을 균형 있게 조절한다.
