# 합성 데이터 생성

합성 데이터는 실제 데이터를 모방하여 인공적으로 생성된 데이터다. 프라이버시 보호, 데이터 부족 해결, 편향 완화 등에 활용된다. GAN을 사용한 이미지 합성, LLM을 활용한 텍스트 데이터 생성, 시뮬레이션 기반 센서 데이터 생성 등이 있다. 합성 데이터의 품질은 다운스트림 태스크 성능으로 평가하며, 실제 데이터와의 분포 유사성이 핵심이다.
