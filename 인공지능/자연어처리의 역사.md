# 자연어처리의 역사

자연어처리(NLP)는 1950년대 기계 번역 연구에서 시작되었다. 초기에는 규칙 기반 접근법이 주류였으나, 1990년대 통계적 방법이 등장하면서 패러다임이 전환되었다. 2013년 Word2Vec의 등장으로 단어 임베딩 시대가 열렸고, 2018년 BERT가 등장하면서 사전 학습 모델의 시대가 시작되었다. 현재는 GPT 시리즈로 대표되는 대규모 언어 모델이 NLP의 거의 모든 영역을 지배하고 있다.
