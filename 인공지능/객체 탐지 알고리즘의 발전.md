# 객체 탐지 알고리즘의 발전

객체 탐지는 이미지에서 물체의 위치와 종류를 동시에 식별하는 작업이다. R-CNN 시리즈는 영역 제안 후 분류하는 2단계 방식이다. YOLO(You Only Look Once)는 이미지를 그리드로 나누어 한 번에 탐지하는 1단계 방식으로 실시간 처리가 가능하다. 앵커 프리 방식의 FCOS, CenterNet은 사전 정의된 앵커 없이 직접 바운딩 박스를 예측한다. 자율주행, 의료 영상, 보안 카메라에서 핵심 기술이다.
