# CNN의 작동 원리

합성곱 신경망(CNN)은 이미지 인식에 특화된 딥러닝 모델이다. 합성곱 레이어에서 필터가 입력 이미지를 스캔하며 특징 맵을 생성한다. 초기 레이어는 에지나 색상 같은 저수준 특징을, 깊은 레이어는 얼굴이나 사물 같은 고수준 특징을 감지한다. 풀링 레이어는 공간 크기를 줄여 계산 효율성을 높이고, 완전연결 레이어에서 최종 분류를 수행한다.
