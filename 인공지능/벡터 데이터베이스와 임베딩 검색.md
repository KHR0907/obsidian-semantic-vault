# 벡터 데이터베이스와 임베딩 검색

벡터 데이터베이스는 고차원 벡터를 효율적으로 저장하고 유사도 검색하는 시스템이다. Pinecone, Weaviate, Milvus, ChromaDB 등이 대표적이다. 텍스트를 임베딩 모델로 변환한 후 코사인 유사도나 유클리드 거리로 검색한다. RAG(검색 증강 생성)에서 관련 문서를 찾는 핵심 인프라로, LLM 애플리케이션에서 필수적인 컴포넌트가 되었다.
