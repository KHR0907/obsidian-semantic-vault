# 어텐션 메커니즘 심화

어텐션 메커니즘은 입력의 특정 부분에 선택적으로 집중하는 기법이다. 쿼리, 키, 밸류의 세 요소로 구성되며, 쿼리와 키의 유사도를 계산하여 밸류에 가중치를 부여한다. 멀티헤드 어텐션은 여러 어텐션 헤드를 병렬로 실행하여 다양한 관점의 정보를 포착한다. 크로스 어텐션은 두 시퀀스 간의 관계를 학습하는 데 사용된다.
