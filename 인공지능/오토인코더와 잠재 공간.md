# 오토인코더와 잠재 공간

오토인코더는 입력 데이터를 저차원의 잠재 공간으로 압축한 후 다시 복원하는 비지도 학습 모델이다. 인코더는 입력을 잠재 벡터로 변환하고, 디코더는 이를 원래 차원으로 복원한다. 변분 오토인코더(VAE)는 잠재 공간에 확률 분포를 부여하여 새로운 데이터를 생성할 수 있게 한다. 이미지 생성, 이상 탐지, 데이터 압축 등에 활용된다.
