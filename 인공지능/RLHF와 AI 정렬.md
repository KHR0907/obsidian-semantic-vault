# RLHF와 AI 정렬

인간 피드백 기반 강화학습(RLHF)은 AI 모델을 인간의 선호에 맞게 조정하는 기법이다. 먼저 인간이 모델 출력을 비교 평가하고, 이를 바탕으로 보상 모델을 학습한다. 그런 다음 PPO 같은 강화학습 알고리즘으로 보상 모델에 맞게 언어 모델을 미세 조정한다. DPO는 보상 모델 없이 직접 선호 학습을 하는 더 간단한 대안이다.
