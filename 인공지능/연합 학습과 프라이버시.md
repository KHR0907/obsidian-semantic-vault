# 연합 학습과 프라이버시

연합 학습은 데이터를 중앙 서버에 모으지 않고 각 디바이스에서 로컬로 모델을 학습하는 분산 학습 방식이다. 각 클라이언트는 로컬 모델 업데이트만 서버에 전송하고, 서버는 이를 집계한다. 차등 프라이버시를 결합하면 개별 데이터를 추론할 수 없도록 노이즈를 추가한다. 의료 데이터, 키보드 예측 등 프라이버시가 중요한 영역에서 활용된다.
