# 배치 정규화의 효과

배치 정규화는 각 레이어의 입력을 정규화하여 내부 공변량 이동을 줄이는 기법이다. 학습 속도를 높이고, 더 높은 학습률을 사용할 수 있게 하며, 정규화 효과도 제공한다. 레이어 정규화는 배치 크기에 의존하지 않아 시퀀스 모델에 적합하고, 그룹 정규화와 인스턴스 정규화도 특정 상황에서 유용하다.
