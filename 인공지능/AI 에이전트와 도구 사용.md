# AI 에이전트와 도구 사용

AI 에이전트는 자율적으로 계획을 세우고 도구를 사용하여 복잡한 작업을 수행하는 시스템이다. 웹 브라우징, 코드 실행, API 호출 등의 도구를 활용한다. ReAct 프레임워크는 추론과 행동을 번갈아 수행하고, AutoGPT와 같은 시스템은 목표를 하위 작업으로 분해한다. 멀티 에이전트 시스템에서는 여러 에이전트가 협력하여 더 복잡한 문제를 해결한다.
