# 그래프 신경망 개요

그래프 신경망(GNN)은 그래프 구조 데이터를 처리하는 딥러닝 모델이다. 노드 간의 관계와 구조 정보를 학습하여 노드 분류, 링크 예측, 그래프 분류 등을 수행한다. GCN은 이웃 노드의 특징을 집계하고, GAT는 어텐션으로 이웃의 중요도를 학습한다. 소셜 네트워크 분석, 분자 속성 예측, 추천 시스템 등에 활용된다.
