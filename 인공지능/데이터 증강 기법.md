# 데이터 증강 기법

데이터 증강은 기존 데이터를 변환하여 학습 데이터를 인위적으로 늘리는 기법이다. 이미지에서는 회전, 반전, 크롭, 색상 변환, 노이즈 추가 등이 일반적이다. Mixup은 두 이미지를 섞어 새로운 학습 샘플을 만들고, CutMix는 이미지의 일부를 다른 이미지로 대체한다. 자연어 처리에서는 역번역, 동의어 치환, 문장 셔플링 등을 사용한다.
