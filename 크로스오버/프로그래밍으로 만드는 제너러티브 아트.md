# 프로그래밍으로 만드는 제너러티브 아트

제너러티브 아트는 알고리즘으로 만들어진 예술이다. Processing과 p5.js는 시각 예술 프로그래밍의 대표적인 도구다. 펄린 노이즈는 자연스러운 흐름과 패턴을 생성하고, L-시스템은 나무나 식물의 구조를 재현한다. NFT 아트 씬에서 아트블록스 같은 플랫폼이 제너러티브 아트를 대중화시켰다. 코드가 붓이 되고, 알고리즘이 창의적 의도를 표현한다.
