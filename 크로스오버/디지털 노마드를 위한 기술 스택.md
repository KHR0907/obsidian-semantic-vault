# 디지털 노마드를 위한 기술 스택

디지털 노마드로 일하려면 안정적인 기술 환경이 필수다. Tailscale VPN으로 어디서든 안전하게 회사 네트워크에 접속하고, VS Code Remote SSH로 원격 서버에서 개발한다. Notion이나 Obsidian으로 지식을 관리하고, Figma와 Linear로 비동기 협업한다. Starlink 위성 인터넷은 오지에서도 연결을 제공한다. 타임존 차이는 비동기 커뮤니케이션 문화로 해결한다.
