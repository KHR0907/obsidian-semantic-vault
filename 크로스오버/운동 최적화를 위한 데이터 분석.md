# 운동 최적화를 위한 데이터 분석

웨어러블 기기와 데이터 분석이 운동 과학을 변화시키고 있다. 심박수 변이도(HRV)는 회복 상태를 추적하고, GPS 데이터는 러닝 효율을 분석한다. 머신러닝 알고리즘이 개인 맞춤형 운동 프로그램을 설계하고, 부상 위험을 예측한다. Strava나 TrainingPeaks 같은 플랫폼은 TSS(훈련 스트레스 점수)를 기반으로 피로와 체력을 모니터링한다.
