# 비타민 D의 역할

비타민 D는 칼슘 흡수와 뼈 건강에 필수적이며, 면역 기능 조절에도 관여한다. 피부가 자외선 B에 노출되면 체내에서 합성되지만, 실내 생활이 많은 현대인은 부족하기 쉽다. 혈중 25(OH)D 수치 30ng/mL 이상이 적절하며, 부족 시 보충제로 하루 1000-2000IU를 권장한다. 지용성 비타민이므로 지방과 함께 섭취하면 흡수율이 높아진다.
