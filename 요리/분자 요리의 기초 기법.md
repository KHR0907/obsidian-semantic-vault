# 분자 요리의 기초 기법

분자 요리는 과학 원리를 활용하여 전통 요리의 형태와 식감을 변화시키는 기법이다. 스페리피케이션은 알긴산나트륨과 염화칼슘의 반응으로 액체를 구슬 형태로 만든다. 젤화는 한천이나 젤라틴으로 액체를 고체로 변환한다. 거품화(에스푸마)는 사이펀으로 가스를 주입하여 소스를 경량 거품으로 만든다. 액체 질소를 이용한 급속 냉동도 대표적이다.
