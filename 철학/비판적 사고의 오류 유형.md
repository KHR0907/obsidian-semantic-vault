# 비판적 사고의 오류 유형

논리적 오류는 형식적 오류와 비형식적 오류로 나뉜다. 형식적 오류는 논증 구조 자체가 타당하지 않은 경우이고, 비형식적 오류는 내용에서 발생한다. 인신공격의 오류는 주장이 아닌 주장자를 공격하고, 허수아비 공격은 상대 주장을 왜곡하여 반박한다. 미끄러운 비탈길 오류는 한 걸음이 연쇄적으로 극단적 결과를 낳는다고 주장한다.
