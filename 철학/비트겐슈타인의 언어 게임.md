# 비트겐슈타인의 언어 게임

후기 비트겐슈타인은 언어의 의미가 사용에서 비롯된다고 주장했다. 언어 게임이란 특정 맥락에서 언어가 사용되는 다양한 방식을 가리킨다. 단어의 의미는 사전적 정의가 아니라 삶의 형식 속에서 결정된다. 가족 유사성 개념은 하나의 공통 본질이 없어도 서로 닮은 특징들로 묶일 수 있음을 보여준다. 사적 언어의 불가능성도 핵심 논점이다.
