# 외계행성 탐색 방법

외계행성은 주로 간접적 방법으로 발견된다. 시선 속도법은 행성의 중력이 항성을 미세하게 흔드는 것을 도플러 효과로 감지한다. 트랜짓법은 행성이 항성 앞을 지날 때 밝기가 감소하는 것을 측정하며, 케플러 우주망원경이 이 방법으로 수천 개의 행성을 발견했다. 직접 촬영법은 코로나그래프로 항성 빛을 차단하고 행성을 직접 관측한다. 제임스 웹 우주망원경은 외계행성 대기 성분 분석을 가능하게 했다.
