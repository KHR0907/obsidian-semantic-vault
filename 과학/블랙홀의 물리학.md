# 블랙홀의 물리학

블랙홀은 중력이 너무 강해 빛조차 탈출할 수 없는 천체다. 사건의 지평선은 탈출이 불가능한 경계면이며, 특이점은 밀도가 무한대인 중심이다. 호킹은 블랙홀이 완전히 검지 않으며 열복사(호킹 복사)를 방출한다고 예측했다. 2019년 이벤트 호라이즌 망원경이 M87 블랙홀의 그림자를 최초로 촬영하여 일반 상대성 이론을 다시 확인했다.
