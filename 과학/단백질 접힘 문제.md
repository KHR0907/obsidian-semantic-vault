# 단백질 접힘 문제

단백질 접힘은 아미노산 서열이 3차원 구조로 접히는 과정이다. 레빈탈의 역설에 따르면 모든 가능한 구조를 탐색하려면 우주의 나이보다 긴 시간이 필요하지만, 실제로 단백질은 밀리초 내에 접힌다. 잘못 접힌 단백질은 알츠하이머, 파킨슨 같은 질병을 유발한다. AlphaFold는 딥러닝으로 단백질 구조를 원자 수준의 정확도로 예측하여 생물학에 혁명을 가져왔다.
