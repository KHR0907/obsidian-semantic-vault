# Semantic Graph 테스트용 Vault

이 vault는 **Obsidian 3D Semantic Graph** 플러그인 테스트를 위해 생성된 더미 데이터입니다.

## 구조

| 클러스터 | 노트 수 | 설명 |
|---------|---------|------|
| 인공지능 | 24 | AI/ML, 딥러닝, NLP 등 |
| 요리 | 20 | 레시피, 발효, 식재료 과학 등 |
| 철학 | 20 | 동서양 철학, 윤리학 등 |
| 여행 | 20 | 세계 여행, 트레킹, 문화 체험 등 |
| 프로그래밍 | 21 | 웹개발, 아키텍처, DevOps 등 |
| 음악과 예술 | 16 | 음악 이론, 미술사, 예술 기법 등 |
| 건강 | 15 | 운동, 영양, 수면, 명상 등 |
| 과학 | 15 | 물리학, 생물학, 천문학 등 |
| 크로스오버 | 15 | 여러 주제를 연결하는 브릿지 노트 |

**총 노트 수: 166**

## 3D Semantic Graph에서 기대하는 결과

- **8개의 뚜렷한 클러스터**가 3D 공간에서 분리되어 나타나야 합니다
- **브릿지 노트**는 클러스터 사이에 위치해야 합니다
- 같은 클러스터 내 노트들은 서로 가깝게 배치되어야 합니다
- 위키링크 `[[]]`와 의미적 유사도가 시각적으로 확인 가능해야 합니다

## 사용법

1. 이 폴더를 Obsidian에서 Vault로 열기
2. 3D Semantic Graph 플러그인 설치
3. OpenAI API 키 설정
4. 임베딩 생성 후 3D 그래프 확인
