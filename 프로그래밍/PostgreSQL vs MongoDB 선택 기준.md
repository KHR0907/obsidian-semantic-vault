# PostgreSQL vs MongoDB 선택 기준

PostgreSQL은 관계형 데이터베이스로 ACID 트랜잭션, 복잡한 조인, 스키마 무결성이 강점이다. 정규화된 데이터 구조에 적합하며 JSON 지원도 좋다. MongoDB는 도큐먼트 기반 NoSQL로, 유연한 스키마와 수평적 확장이 장점이다. 빠른 프로토타이핑과 비정형 데이터에 적합하다. 일관성보다 가용성이 중요하면 MongoDB, 무결성이 중요하면 PostgreSQL이다.
