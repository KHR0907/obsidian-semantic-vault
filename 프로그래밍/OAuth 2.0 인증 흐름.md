# OAuth 2.0 인증 흐름

OAuth 2.0은 제3자 애플리케이션이 사용자 자원에 접근할 수 있도록 하는 인가 프레임워크다. Authorization Code 흐름은 서버 사이드 앱에 적합하며 가장 안전하다. PKCE 확장은 모바일과 SPA의 보안을 강화한다. 액세스 토큰은 짧은 수명으로 발급하고, 리프레시 토큰으로 갱신한다. OpenID Connect는 OAuth 위에 인증(identity) 레이어를 추가한다.
