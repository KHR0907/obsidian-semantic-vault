# 리액트 상태 관리 변천사

리액트의 상태 관리는 setState에서 시작해 Redux가 전역 상태 관리의 표준이 되었다. Redux는 단일 스토어와 불변 상태, 순수 리듀서를 원칙으로 한다. 이후 Context API가 간단한 전역 상태에, MobX가 반응형 접근에 대안을 제시했다. Zustand와 Jotai는 더 가벼운 대안이며, React Query는 서버 상태를 별도로 관리하는 새로운 패러다임을 열었다.
