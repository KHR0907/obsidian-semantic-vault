# 자바스크립트 비동기 처리 진화

자바스크립트의 비동기 처리는 콜백에서 시작했다. 콜백 지옥을 해결하기 위해 Promise가 도입되었고, then 체이닝으로 가독성이 개선되었다. ES2017의 async/await는 비동기 코드를 동기 코드처럼 작성할 수 있게 한다. 에러 처리는 try/catch로 하며, Promise.all과 Promise.allSettled로 병렬 처리를 관리한다.
