# 도커 컨테이너 vs 가상 머신

가상 머신(VM)은 하이퍼바이저 위에서 전체 OS를 실행하므로 완전한 격리를 제공하지만 무겁다. 도커 컨테이너는 호스트 OS의 커널을 공유하며 프로세스 수준의 격리를 제공한다. 컨테이너는 수 초 내에 시작되고 메모리 사용량이 적다. Dockerfile로 이미지를 정의하고 docker-compose로 멀티 컨테이너 환경을 관리한다.
