# 알고리즘 시간 복잡도 정리

Big O 표기법은 알고리즘의 최악 시간 복잡도를 나타낸다. O(1)은 상수 시간(해시 테이블 조회), O(log n)은 로그 시간(이진 검색), O(n)은 선형 시간(배열 순회)이다. O(n log n)은 효율적인 정렬 알고리즘(머지소트, 퀵소트)의 평균 복잡도다. O(n²)은 중첩 루프(버블 정렬), O(2ⁿ)은 지수 시간(피보나치 재귀)이다.
