# 웹 보안의 기초

XSS(크로스 사이트 스크립팅)는 악성 스크립트를 웹 페이지에 주입하는 공격이다. 입력 값 이스케이핑과 CSP 헤더로 방어한다. SQL 인젝션은 SQL 쿼리에 악성 코드를 삽입하는 것으로, 파라미터화된 쿼리로 방지한다. CSRF는 인증된 사용자의 권한으로 원치 않는 요청을 보내는 공격이며, 토큰 기반 검증으로 막는다. HTTPS는 전송 구간 암호화의 기본이다.
