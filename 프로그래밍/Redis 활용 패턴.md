# Redis 활용 패턴

Redis는 인메모리 키-밸류 스토어로 캐시, 세션 관리, 메시지 브로커 등에 활용된다. 문자열, 해시, 리스트, 셋, 정렬 셋 등 다양한 데이터 구조를 지원한다. 캐시 무효화 전략으로 TTL 설정, 쓰기 시 갱신(write-through), 지연 갱신(lazy loading)이 있다. Pub/Sub으로 실시간 메시징을, Sorted Set으로 리더보드를 구현한다.
