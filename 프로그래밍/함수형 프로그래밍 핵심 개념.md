# 함수형 프로그래밍 핵심 개념

함수형 프로그래밍의 핵심은 순수 함수와 불변 데이터다. 순수 함수는 같은 입력에 항상 같은 출력을 반환하고 부작용이 없다. map, filter, reduce는 컬렉션을 변환하는 기본 연산이다. 고차 함수는 함수를 인자로 받거나 반환하는 함수다. 커링은 다중 인자 함수를 단일 인자 함수의 체인으로 변환한다.
