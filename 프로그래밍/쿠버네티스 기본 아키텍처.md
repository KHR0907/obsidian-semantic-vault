# 쿠버네티스 기본 아키텍처

쿠버네티스는 컨테이너 오케스트레이션 플랫폼이다. 마스터 노드의 API 서버, 스케줄러, 컨트롤러 매니저가 클러스터를 관리한다. 워커 노드에서 kubelet이 파드(Pod)를 실행한다. 파드는 하나 이상의 컨테이너를 포함하는 최소 배포 단위다. 서비스(Service)는 파드에 안정적인 네트워크 접점을 제공하고, 인그레스는 외부 트래픽을 라우팅한다.
