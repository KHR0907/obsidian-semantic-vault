# 파이썬 제너레이터와 이터레이터

이터레이터는 __iter__와 __next__ 메서드를 구현하는 객체로 순회 가능한 데이터를 표현한다. 제너레이터는 yield 키워드를 사용하여 이터레이터를 간결하게 만드는 함수다. 값이 필요할 때마다 하나씩 생성하므로 메모리 효율이 뛰어나다. 대용량 파일 처리나 무한 시퀀스에 유용하다. 제너레이터 표현식은 리스트 컴프리헨션의 지연 평가 버전이다.
