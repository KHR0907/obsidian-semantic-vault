# 파이썬 데코레이터 패턴

데코레이터는 함수나 클래스를 수정하지 않고 기능을 추가하는 파이썬의 강력한 기능이다. @login_required처럼 인증 체크를, @cache로 메모이제이션을 구현할 수 있다. 내부적으로 데코레이터는 함수를 인자로 받아 새 함수를 반환하는 고차 함수다. functools.wraps를 사용하면 원래 함수의 메타데이터가 보존된다.
